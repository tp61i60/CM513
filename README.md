`bookstore.py` 是使用 streamlit 製作的書店電商網站，大家可以以這個檔案為基礎修改成自己的網站。

這個網站主要分為 6 個頁面，分別為：

1. 登錄頁面
2. 商品總覽
3. 購物車
4. 結帳
5. 留言板
6. 購買歷史紀錄

我們目前的登錄帳號有兩個，分別為：

1. 帳號：`jsmith` 密碼：`abc`
2. 帳號：`rbriggs` 密碼：`def`

若想新增帳號可以更新 `config.yaml`，我們將客戶的登錄資訊都寫在這個檔案中。
